# System Architecture

## Data Flow

User
↓
Input Module
↓
Preprocessing Module
↓
Rule Engine
↓
Risk Calculator
↓
Report Generator
↓
User

## Modules

### Input Module
Receives email subject and body.

### Preprocessing Module
Converts text into standardized format.

### Rule Engine
Detects phishing indicators.

### Risk Calculator
Calculates phishing score.

### Report Generator
Displays final report.
